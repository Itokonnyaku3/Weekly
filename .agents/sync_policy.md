# Antigravity 自動同期プロトコル

本リポジトリにおいて、Antigravity（AI開発アシスタント）が遵守すべき Git 運用のルールです。

## 1. 原則
Antigravity は、ファイル修正を完了し、かつそれが論理的にひとまとまりの変更（プロジェクトの動作を壊さない状態）であると判断した場合、自動的にコミットおよびプッシュを実行します。

## 2. 定型コマンド
以下の手順で同期を行います：
1. `git add .` (または修正対象のファイル)
2. `git commit -m "Auto sync by Antigravity: [変更の要約]"`
3. `git push origin main`

## 3. 注意事項
- ユーザーから明示的に「コミットを控えてほしい」旨の指示があった場合は、それに従います。
- 大きな変更や破壊的な変更を行う場合は、プッシュ前にユーザーに確認を求めます。
