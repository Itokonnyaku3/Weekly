⚙️ 設定

💾 ファイル自動保存

ローカルの data.json に直接保存します。Chrome / Edge のみ対応。

確認中...

※ 選択したファイルは次回以降も自動的に使用されます（ブラウザの許可が必要）。localStorage も同時に更新します。

📦 GitHub 同期設定

※ repo 権限が必要です。トークンはブラウザの localStorage にのみ保存されます。

同期ステータス: 未設定

📂 データのバックアップ・復元 (JSON)

※ 読み込み（復元）を行うと、現在のデータは上書きされますのでご注意ください。

🕒 GitHubへ日次バックアップ（backups/ に過去3日分を保持・起動時に自動／GitHub同期設定が必要）

🛠️ メンテナンス

画像を GitHub フォルダへ移行

ノート内に埋め込まれた base64 画像を検出し、GitHub リポジトリの images/ フォルダにアップロードしてリンクに置換します。これにより data.json を大幅に軽量化できます。

※ 実行前に GitHub 同期設定が完了していることを確認してください。
